# BloodLink

Monorepo du projet BloodLink.

## Structure

- `backend/` : API FastAPI
- `mobile/` : application Flutter mobile
- `admin_web/` : interface d'administration Flutter web
- `docs/` : documentation projet
- `scripts/` : scripts utilitaires

## Démarrage du projet

Le dépôt est préparé pour un développement local sans Docker.

### Étapes prévues

- initialiser le backend dans `backend/`
- initialiser l'application mobile dans `mobile/`
- initialiser l'application admin web dans `admin_web/`
- centraliser la documentation active dans `docs/`
